== Description ==

Automatically pull your latest tweets into your widget areas and have them displayed within a slick news-scroller styled container.

== Installation ==

1. Upload `Twitter_Scroller` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag the 'Twitter Feed' widget into your widgets area, set title, username and number of tweets to show, then enjoy!
